# 💻 Hoja de Vida Web – Jonathan Camilo Herrera Camacho

Sitio web personal creado como proyecto académico para la asignatura de **Desarrollo Web** en la Universidad Compensar.

## 🔗 Ver sitio en línea

> **[https://carlosmora.github.io/hoja-de-vida](https://carlosmora.github.io/hoja-de-vida)**

---

## 📁 Estructura del proyecto

```
hoja_de_vida/
│
├── index.html              → Página principal (inicio + sobre mí)
├── css/
│   └── styles.css          → Hoja de estilos general del sitio
└── pages/
    ├── experiencia.html    → Experiencia laboral y certificaciones
    ├── proyectos.html      → Proyectos académicos y personales
    └── contacto.html       → Formulario y datos de contacto
```

## 🛠️ Tecnologías usadas

- **HTML5** – Estructura semántica del sitio
- **CSS3** – Estilos, diseño responsive, variables CSS
- **Google Fonts** – Tipografías Fira Code y Nunito
- **GitHub Pages** – Hosting gratuito del sitio

## 📖 Cómo usar (localmente)

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/carlosmora/hoja-de-vida.git
   ```
2. Abrir `index.html` en el navegador.

No requiere instalación de dependencias, todo es HTML y CSS puro.
